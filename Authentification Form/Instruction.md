# Просто http сервер (и никаких проблем)

## документация
https://www.npmjs.com/package/http-server


## установить необходимую библиотеку
npm install http-server -g


## запуск http сервера 
http-server


## запуск https сервера
http-server -S -C cert.pem


## адрес сервера
выводится в консоли
